### 3. Pattern SVGs (African Designs)
Create or download these patterns:

#### Adire Pattern (Yoruba)
- **File**: `public/patterns/adire.svg`
- **Type**: Repeating pattern
- **Colors**: Indigo blue on white
- **Usage**: Background patterns
- **How to create**: Use Illustrator or find free SVG patterns

#### Uli Pattern (Igbo)
- **File**: `public/patterns/uli.svg`
- **Type**: Flowing lines
- **Colors**: Black on transparent
- **Usage**: Decorative borders
- **How to create**: Draw in Illustrator or commission artist

#### Geometric Pattern (Hausa)
- **File**: `public/patterns/geometric.svg`
- **Type**: Islamic-inspired geometry
- **Colors**: Blue on transparent
- **Usage**: Grid backgrounds
- **How to create**: Use pattern generators online

#### Kente Strip
- **File**: `public/patterns/kente-strip.svg`
- **Type**: Horizontal colorful strip
- **Colors**: Red, yellow, green, gold, blue
- **Usage**: Progress bars, dividers
- **How to create**: Simple rectangles in Illustrator

**Free Pattern Resources:**
- https://www.svgbackgrounds.com/

## üìä Image Specifications Summary

### Formats
- **Photos**: JPG (compressed, 80-90% quality)
- **Graphics**: PNG (with transparency)
- **Patterns**: SVG (scalable)
- **Icons**: SVG or PNG

### Sizes
```
Hero Images:      1920x1080px (16:9)
Course Cards:     800x600px (4:3)
Event Cards:      600x400px (3:2)
Team Photos:      400x400px (1:1)
Alumni Photos:    300x300px (1:1)
Thumbnails:       200x200px (1:1)
Icons:            64x64px (1:1)
Patterns:         SVG (any size)
```

### Optimization
- Compress all images before upload
- Use WebP format for better performance
- Lazy load images below the fold
- Use responsive images (srcset)

**Tools:**
- https://tinypng.com/ (compression)
- https://squoosh.app/ (compression & format conversion)
- https://imageoptim.com/ (Mac app)
